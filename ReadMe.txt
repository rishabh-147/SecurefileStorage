The libraries are added in two folders
1. Download both the zip files
2. Unzip them
3. Create a folder "Lib" and put files from both the folders into this one folders.
4. Create a folder "venv" 
5. Add the Library file  "Lib" and  "Scripts" and also "pyvenv.cfg" into venv
6. save this folder into the "SecurefileStorage" folder of the project
